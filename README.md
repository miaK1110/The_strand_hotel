### サイトの説明

The strand hotel は海辺にあるホテル&レストランをイメージして XD でデザインから作成した Web サイトです。
ハンバーガーメニューによるアンカーリンク、画像スライダー、文字のフェードインなどを実装し
レスポンシブ Web デザインにし、パッケージ管理には npm を、タスク管理には Gulp を使いました。
また、CSS 設計には FLOCSS 設計を採用しました。
Google Chrome、Safari、IE11、FireFox から作った Web サイトを見ることでそれぞれのブラウザによる見え方の違いの勉強にもなりました。

### 苦労したこと

- レスポンシブコーディングにより高さが可変するスティッキーヘッダーメニューを実装した際、ヘッダーの高さ分
  アンカーリンクの位置がずれてしまうので jQuery で高さを取得し高さ分マイナスをしました。

- FLOCSS 設計で破綻しないコードを書くにはどうすればいいのか、修正しやすく他の人から見ても
  見やすいコードを書くにはどうすればいいのかに悩みました。
