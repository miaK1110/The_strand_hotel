### サイトの説明

The strand hotelは海辺にあるホテル&レストランをイメージしてXDでデザインから作成したWebサイトです。
ハンバーガーメニューによるアンカーリンク、画像スライダー、文字のフェードインなどを実装し
レスポンシブWebデザイン、パッケージ管理にはnpmを、タスク管理にはGulpを使いました。
また、CSS設計にはFLOCSS設計を採用しました。
Google Chrome、Safari、IE11、FireFoxから作ったWebサイトを見る・修正することでそれぞれのブラウザによる見え方の違いの勉強にもなりました。


### 苦労したこと

- レスポンシブコーディングにより高さが可変するスティッキーヘッダーメニューを実装した際、ヘッダーの高さ分
アンカーリンクの位置がずれてしまうのでjQueryで高さを取得し高さ分マイナスをしました。

- FLOCSS設計で破綻しないコードを書くにはどうすればいいのか、修正しやすく他の人から見ても
見やすいコードを書くにはどうすればいいのかに悩みました。
